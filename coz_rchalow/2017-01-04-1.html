＊ RC-S330をLubuntu_16.04で使う [RC-S330][Lubuntu][R731]

google先生だと2014年以前の情報しか見つからなかったので書いておく。
うちの環境はLubuntu_16.04_LTS(64bit)

sudo apt-get install libnfc-dev
して再起動。
マイナンバーカードを載せたままUSB接続で認識された。
実際インストされたのは以下
Inst libusb-dev (2:0.1.12-28 Ubuntu:16.04/xenial [amd64])
Inst libnfc5 (1.7.1-4 Ubuntu:16.04/xenial [amd64])
Inst libnfc-dev (1.7.1-4 Ubuntu:16.04/xenial [amd64])
　
nfc-listを試したいなら
sudo apt-get install libnfc-bin
が必要みたい。
　　
gsuica、nfcpy あたりをインストすれば少し遊べそう。
