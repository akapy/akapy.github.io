＊ 早くも確定申告が終わった！ [RC-S330][確定申告][e-Tax]

e-Taxだと正月休み明けから申告できるのね。知らなかった。
Linuxで申告書作成して、Win7なIE11から送信するのが一番シンプルみたい。
うちはLubuntu16.04とWin7proのデュアルブート環境
　
　　
以下、ブラウザと必要最低限のインストで終わらせるための覚え書き

・ICカードリーダーを使えるようにする
　　
・WinのIE11を起動
・http://www.e-tax.nta.go.jp/ の [e-Taxソフト(WEB版)(ログイン)] ボタンを押す。
・利用環境が合うように指示に従う。
　　
・開始届出書作成・提出
・利用者識別番号等の取得
　　
　　
・Linuxのブラウザで確定申告書等作成コーナーを開く
・e-Tax用の申告書を作成・保存：xtxファイル２個？
　　
　　
・WinのIE11を起動
・確定申告書等作成コーナーから送信

＊ RC-S330をLubuntu_16.04で使う [RC-S330][Lubuntu][R731]

google先生だと2014年以前の情報しか見つからなかったので書いておく。
うちの環境はLubuntu_16.04_LTS(64bit)

sudo apt-get install libnfc-dev
して再起動。
マイナンバーカードを載せたままUSB接続で認識された。
実際インストされたのは以下
Inst libusb-dev (2:0.1.12-28 Ubuntu:16.04/xenial [amd64])
Inst libnfc5 (1.7.1-4 Ubuntu:16.04/xenial [amd64])
Inst libnfc-dev (1.7.1-4 Ubuntu:16.04/xenial [amd64])
　
nfc-listを試したいなら
sudo apt-get install libnfc-bin
が必要みたい。
　　
gsuica、nfcpy あたりをインストすれば少し遊べそう。
