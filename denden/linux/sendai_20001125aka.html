ＴＬＵＣ大インストール大会 < 2000.11.25 >

みなさん、Linuxって見たことがありますか？触ったことは？新聞、雑誌 TVなどで、だれもが一度は目にしたLinux。チャレンジした方も多いでしょう。 でも、Windows に次ぐユーザー数を誇ると言われている Linux ですが、実際にイン ストールしてみようとして挫折した、 あるいはインストールはしたものの、何 に使えばよいかわからない、さらにはこの機会に Linux に触れてみたい、など といろいろな感じ方をされていると思います。 　そこで、 TLUC-東北 Linux ユーザー連合会- では、下記のスケジュールで 大インストール大会を行います。

  • まだ自分のPCに Linux をインストールしたことがない
  • インストールをしてみたけどうまくいかない
  • どうにかインストールできたけど, どうやって使うのかよくわからない.

といった方々を主な対象とし, みんなで一緒に Linux をインストールし, お互いに教え合いましょうという企画です. ちょっとだけ経験のある(みんな素人ではありますが) TLUC のメンバーが, できるだけお手伝いしたいと考えています。 皆さんのご参加をお待ちしております。

日時 : 8 月 21 日(土) 10:00〜15:30
場所 : 仙台市勤労者保養所 茂庭荘
  • 仙台市太白区茂庭字人来田１４３−３
内容 :

「 Linux のインストールと基本操作を学ぼう！ 」

  • インストールに使用するディストリビューションは、現時点では、 RedHat 系の Vine Linux 2.0 を予定しています。なお、参加者には、 Vine Linux 2.0 FTP 版を CD-R にて配布することも検討しています。
  • Linux のインストール, X-Window System の設定, シェルやエディタなどの使い方などについて, 簡単に実習を行います.
参加費 : インストールに利用するメディア, 配布資料の実費として 500円 を予定しています.
幹事 : TLUC 仙台 お手伝い係(代表)
定員 : 会場には 60 名ほど入場可能ですが, お手伝いするメンバーの 人数との関係上, インストール希望の方の定員は 20 名程度に 限らせていただく予定です. 見学のみの方は特に定員は設けませんが, 出席予定人数の確認のため, できるだけ事前に参加申し込みをお願いします.
注意 :
  • インストールを希望される方には, ご自分でPCを持ち込んでいただ きます. モニタを含む本体一式をご持参ください. なお, インストールをできるだけ円滑に行うため, 申し込み時に は必ず機種の詳細について, アンケートの記入をお願いいたします.
  • 対象機種は, PC/AT 互換機 (DOS/V マシン)に限定させていた だきます.
  • NIC (ネットワークインタフェースカード)をお持ちの方は, なる べくご持参下さい.
  • 諸般の事情でインストールが達成できない場合も考えられますが, その点についてはあらかじめご理解下さい.
  • 本インストール大会での作業で, 万一データ消失やハードの 故障など, 参加者にとっての不利益が発生しても, 主催者 は一切責任を持ちません. 予め必ずご理解下さい.
応募締切 ： インストール希望の方の〆切は8/20(金)17:00までとします。 なお、定員になりしだい締め切らせていただきます。
