■ e-Taxは残念です [ひとりごと]

tag: [ひとりごと]

e-Taxでも使って確定申告してみるかな、と思ったが

Windowsじゃないと「開始届出書」オンライン提出できないのか？
嫌々Windows立ち上げて申し込もうとしたらダメ。
あらためて利用環境をちゃんと確認したら
(1)　ＯＳ

　オンラインで利用開始（変更）届出を行うことができるＯＳの種類及びバージョンは、次のとおりです。

    * 　Microsoft Windows 2000 Professional
    * 　Microsoft Windows XP

(2)　ブラウザ

　オンラインで利用開始（変更）届出を行うことができるWWWブラウザの種類及びバージョンは、次のとおりです。

    * 　Microsoft Internet Explorer6.0以降

(3)　Adobe Reader

　作成コーナーを利用して開始届出書の作成を行うには、ご利用の環境にAdobe Readerのバージョン7.0.1以上が必要です。
　なお、Adobe Readerのバージョンが7.0.0以下の場合、次のような不具合が発生することが報告されています。

■ 今朝のおかしな夢 [ひとりごと]

tag: [ひとりごと]

ながれでまっこ宅へ行った。どうして行ったのかは思い出せない。
見たことのない家だった。
家族や知合いが帰ってきて騒がしくなってきたので外に出た。
そこはホームセンターみたいな店になってた。ガソリンも売ってたような？
風呂上がりの博ちゃんと一服。
古山商店に行った。故障した車を引き取りに。
店は改装されて邦(くに)の住処になってた。小売業はやめてた。オーディオとかいろんなものがあった。
邦はラーメンが食いたいと言った。
博ちゃんの車はとなりの空き地にあった。ウイングロードか何か。
オイの軽自動車は見つからなかった。
邦が言うには新谷くんがどこかに移動してくれたとか。
目の前に七輪で焼き鳥が焼かれていた。
焦げそうなのでひっくり返した。
新谷くん登場。車の移動場所は事務所だとか。
軍司くんが大川橋から飛び込もうとしてる。度胸試しらしい。

支離滅裂で変な夢だった。
でも以前この前編を見たような感じがする。
邦の家に車を置いてきた夢。

■ 土曜の夜はJAZZ STRUTTIN' [ひとりごと]

tag: [ひとりごと]

今週も土曜日だというのに仕事。
昨日の進行具合だと15時ぐらいに上がれるかな、という感じ。
早く終わったらプールにでも行くか。
終わってみたら１８時。
とりあえず旭小屋に帰る。オリーブに電話する。

ポカンと暇になってしまった。
危うく飲みに出かけそうになったが考え直す。
「酒のあつみ」で蔵王スターのロゼを購入。
ほか弁でデラックス弁当。ちょっと贅沢したつもりだが別なのにすればよかった…。
家で弁当をつまみにちびちびやる。
JAZZ STRUTTIN' を聞きながら。

土曜日の夜はどうしても飲む用事が多い。
ＦＭ聞きながら飲める店があってもいいんだけどね。
