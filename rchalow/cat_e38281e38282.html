■ debianとNFCスマホで確定申告 [めも][Linux][確定申告]

tag: [めも][Linux][確定申告]

win10やwin11を使わずに確定申告ができたというwebページを見かけた
　https://fltax.net/linux-etax/

真似するしかない！
　
floorpブラウザをインストールしたが、

debian では sudo apt install floorp すると「未知の圧縮問題」が発生
以下を参考にインスト
　https://zenn.dev/nagomiso/articles/59dc56d37a65d4
しかし、floorpのユーザーエージェント変更方法が変わったみたいであきらめる
　
chromeでユーザーエージェントを変更して同様にやってみた
　https://engrmemo.jp/archives/ubuntu-browser-user-agent/

申告書作成コーナーの独自画面(ブラウザのメニューなどがない画面)が出てから、
[F12]押して、縦点３個メニュー押して、[More tools]選んで、[Network conditions]選んで、
User agentのチェックボックス外してChrome--Windowsを選択
　
　
　
ICカードリーダーは使わないので
スマホでQRコードを読み込んでログイン
　
　
この方法で確定申告書作成コーナーで送信まで完了できた！

SONY RC-S300は使わない

必要なもの：debianなパソコン、NFCなスマホ、マイナンバーカード

売上データ、源泉徴収票、経費データ、

国民健康保険の支払額

■ 回転チェアー DC-46KP(MB2) の修理 [DIY][めも]

tag: [DIY][めも]

茶の間のダイニングこたつの椅子が回転しなくなった。
床を見ると黒い金属粉で黒くなってる。。
DC-46KP(MB2)というラベルだけしか分からない。
ググったら同じ症状をDIYで修理してるひとがいた。
　[[https://edit-anything.com/blog/rotary-chair.html]]
　
似た部品を探してみたら「7Akiranoai 回転盤 160x25mm」が現物に近いようだ。
amazonで発注1,800円程度
　
２日後に届いた。
アーレンキー(六角レンチ)だけあれば外すことはできる。
壊れたものと届いたものを比較すると厚みがちょっと違うのと、ダルマ加工の有無
そのままだとネジ８箇所中２箇所が入らないかった。
金ヤスリで長穴を広げる程度の追加工で無事に取り付け完了
復活しました！(嬉)
